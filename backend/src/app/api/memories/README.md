# Memory Routes

## Endpoints

- `GET /api/memories`
- `POST /api/memories`
- `GET /api/memories/[id]`
- `PATCH /api/memories/[id]`
- `DELETE /api/memories/[id]`
- `POST /api/memories/extract`
- `POST /api/memories/extract/confirm`
- `GET /api/memories/[id]/connections`
- `POST /api/memories/[id]/connections`
- `GET /api/memories/[id]/graph`
- `DELETE /api/memories/connections/[id]`

## Purpose

Structured memory CRUD plus extraction/graph workflows.
