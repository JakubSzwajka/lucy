# Agent Config Routes

## Endpoints

- `GET /api/agent-configs`
- `POST /api/agent-configs`
- `GET /api/agent-configs/[id]`
- `PUT /api/agent-configs/[id]`
- `DELETE /api/agent-configs/[id]`

## Purpose

CRUD for agent configurations (personas, tool sets, delegation rules).
