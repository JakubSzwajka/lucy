# Obsidian Integration

Adapter for Obsidian Local REST API.

## Public API

- `obsidianIntegration`
- `ObsidianClient`
- note types: `NoteInfo`, `NoteContent`

## Use It Like This

Used by note/memory tool modules that persist markdown knowledge in an Obsidian vault.
Requires `OBSIDIAN_API_KEY` and optional `OBSIDIAN_BASE_URL`.
