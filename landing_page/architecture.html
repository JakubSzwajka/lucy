One Function

The branching point is one flag

Streaming mode returns an SSE stream. Non-streaming mode loops generateText() until no more tool calls remain.

chat.service.ts
async runAgent(agentId, userId, messages, options) { // Same for chat, delegation, and reflection const context = await prepareChat(agentId, userId, options) // → resolves: agentConfig → model + tools + systemPrompt if (options.streaming) { // Chat mode: SSE stream back to frontend return streamText({ model, messages, tools, onStepFinish: persist each step to DB, onFinish: finalize + maybeAutoReflect(), }) } else { // Delegation + reflection: loop until done for (let turn = 0; turn < maxTurns; turn++) { const result = await generateText({ model, messages, tools }) persistSteps(result) if (!result.toolCalls.length) break // done } return finalResult } }
Why this matters
A delegate agent has the same capabilities as the root agent. A reflection agent is just another agent with memory tools. There's no special-case code for delegation or reflection — they're just agent configs pointed at runAgent().
Config Resolution

Data Model

Sessions, agents, items

A session owns a root agent. Delegation creates child sessions. Every step is persisted as an item — messages, tool calls, tool results, reasoning.

Session
Container. Has rootAgentId, parentSessionId for delegation chains.
→
Agent
Runtime instance. Linked to AgentConfig. Status: pending → running → completed.
→
Items
Polymorphic. message | tool_call | tool_result | reasoning. Ordered by creation.
Delegation is just sessions pointing at sessions
When a parent agent delegates, a child session is created with parentSessionId linking back. The child has its own agent, its own items, its own config. The result flows back as a tool_result in the parent's item stream.
Reflection is just a session with memory tools
When token count crosses the threshold, a reflection session is created using the user's reflectionAgentConfigId. The transcript becomes the user message. The agent runs with memory tools and decides what to persist. Fire-and-forget.
