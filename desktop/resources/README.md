# Resources

App icons and logos used by Electron for window icons and installers.

## Files

| File | Purpose |
|------|---------|
| `icon.icns` | macOS app icon |
| `icon.ico` | Windows app icon |
| `icon.png` | Linux app icon / fallback |
| `cat_logo_1.png` | Logo variant 1 |
| `cat_logo_1_cropped.png` | Logo variant 1 (cropped) |
| `cat_logo_2.png` | Logo variant 2 |
| `cat_logo_2_cropped.png` | Logo variant 2 (cropped) |
